# To Do App

## Quick Start

Create a `.env.local` file and add the api key:

```
REACT_APP_API_KEY=XXXX_API_KEY_HERE
```

```bash
git clone https://github.com/serpient/to-do-app.git
cd to-do-app
yarn install

# Start application
yarn start

# Jest Test
yarn run test

# Cypress E2E test
yarn run test:e2e
```
